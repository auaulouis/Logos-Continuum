# Screenshots

This page shows the current Logos Continuum workflows in practice.

## Home page

### Upload and parse documents
![Home upload and parse](./home-upload-parse.png)

- Use the Home search bar for quick navigation to Query.
- Upload one or more `.docx` files to parse/index cards.
- Review parse status, timing, and throughput details.


### Manage indexed/uploaded documents
![Home manage documents](./home-manage-documents.png)

- Open from Settings â†’ **Manage Documents**.
- Search, select, and delete from index and/or uploaded folder.
- Use bulk select/delete for multi-document cleanup.

## Query page

### Search results
![Query search results](./query-search-results.png)

- Run standard text search from the top search bar.
- Use inline citation syntax: `cite:<text>` (example: `nuclear deterrence cite:brookings`).
- Switch between **Tag Matches** and **Paragraph Matches**.
- Navigate pages with **Previous / page number / Next**.

### Card detail and editing
![Query card detail and editing](./query-card-detail-editing.png)

- Select a result to open full card detail.
- Use **Edit**, **Copy**, and **Export Saved Edits (N)** actions.
- Apply formatting in editor mode (highlight, bold, underline, italic).
- Save edits locally and export combined saved edits to `.docx`.
